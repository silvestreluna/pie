## Getting Started
``` 
    $git clone https://github.com/
    nss-evening-cohort-9/pie/
```

## Prerequisites  
```
    $npm install -g http-server
```

## Running
Browse to pie directory and run http
server.

```
    $ hs -p 5000
```
Then, in your browser, Copy and Paste "localhost:5000".
